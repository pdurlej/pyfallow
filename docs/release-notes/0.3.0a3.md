# fallow-py 0.3.0a3 / fallow-py-mcp 0.1.0a3

`0.3.0a3` is the public rename alpha.

## Highlights

- Public package name: `fallow-py`.
- Public MCP package name: `fallow-py-mcp`.
- Canonical console scripts: `fallow-py` and `fallow-py-mcp`.
- Canonical config file/table: `.fallow-py.toml` and `[tool.fallow_py]`.
- Legacy `pyfallow`, `pyfallow-mcp`, `.pyfallow.toml`, and `[tool.pyfallow]`
  remain available during the alpha migration with deprecation warnings.
- Forgejo and GitHub repositories use `pdurlej/fallow-py`.

## Agent Integrations

- Claude Code skill source now lives at `examples/claude-skill/fallow-py-cleanup/`.
- Cursor rule source now lives at `examples/cursor-rules/fallow-py.mdc`.
- The MCP namespace remains `pyfallow` in 0.3.x for compatibility with existing
  tool references; the package and command names are `fallow-py-mcp`.
- `safe_to_remove` returns a structured result with `classifications` and
  `unrecognized` so stale fingerprints cannot be confused with current findings.

## Compatibility

This alpha is source-compatible with the previous `pyfallow` import path and CLI
aliases. New integrations should use `fallow-py` names.

## Verification

Before release, run:

```bash
python -m compileall -q src tests mcp/src mcp/tests
python -m pytest -q
python -m pytest -q mcp/tests
PYTHONPATH=src python -m fallow_py analyze --root . --fail-on warning --min-confidence medium
PYTHONPATH=src:mcp/src python -m fallow_py analyze --root mcp --fail-on warning --min-confidence medium
python -m build
(cd mcp && python -m build)
python -m twine check dist/* mcp/dist/*
```
